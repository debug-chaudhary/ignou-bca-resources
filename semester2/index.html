[span_11](start_span)

1.3 COMPUTER GENERATIONS

[span_29](start_span)

The period, during which the evolution of computer took place, can be divided into five distinct phases known as Generations of Computers.[span_29](end_span)

    [span_30](start_span)
  • First Generation Computers (1940-1956): Characterized by the use of vacuum tubes.[span_30](end_span) [span_31](start_span)They were bulky, required large rooms for installation, and emitted a lot of heat.[span_31](end_span) [span_32](start_span)Examples include ENIAC, EDVAC, and UNIVAC-1.[span_32](end_span)
    • [span_33](start_span)[span_34](start_span)
  • Second Generation Computers (1956-1963): Used transistors instead of vacuum tubes.[span_33](end_span)[span_34](end_span) [span_35](start_span)This made computers smaller, faster, more powerful, and less expensive.[span_35](end_span) [span_36](start_span)High-level programming languages like FORTRAN, ALGOL, and COBOL were introduced.[span_36](end_span)
    • [span_37](start_span)
  • Third Generation Computers (1964-1971): Characterized by the use of Integrated Circuits (ICs) or "chips".[span_37](end_span) [span_38](start_span)This led to smaller size, lower cost, larger memory, and higher processing speed.[span_38](end_span)
    • [span_39](start_span)
  • Fourth Generation Computers (1971-Present): Used microprocessors, which contain all the circuits for arithmetic, logic, and control functions on a single chip.[span_39](end_span) [span_40](start_span)[span_41](start_span)This led to the development of microcomputers and personal computers.[span_40](end_span)[span_41](end_span)
    • [span_42](start_span)
  • Fifth Generation Computers (Present and Beyond): Based on Artificial Intelligence (AI), with applications like voice recognition.[span_42](end_span) [span_43](start_span)[span_44](start_span)These computers focus on parallel processing and aim to communicate in natural language.[span_43](end_span)[span_44](end_span)

3.2 MEMORY AND STORAGE DEVICES

[span_80](start_span)

Memory is an important component of a computer where all the data and information are stored in the form of binary digits.[span_80](end_span) [span_81](start_span)The storage devices are ranked based on access time, storage capacity, and cost-per-bit.[span_81](end_span)

Memory is broadly categorized into two types:

  • Primary or Main Memory: Also called semiconductor memory (e.g., RAM, ROM). [span_82](start_span)[span_83](start_span)It has faster access time, smaller storage capacity, and higher cost per bit.[span_82](end_span)[span_83](end_span) [span_84](start_span)It is a volatile memory.[span_84](end_span)
  • Secondary or Auxiliary Memory: Used for bulk storage (e.g., magnetic disks, optical disks). [span_85](start_span)[span_86](start_span)It is slower, has higher capacity, and lower cost per bit.[span_85](end_span)[span_86](end_span) [span_87](start_span)It is non-volatile.[span_87](end_span)
3.2.1 Semiconductor (Main) Memory
    [span_88](start_span)
  • RAM (Random Access Memory): The read and write memory of a computer.[span_88](end_span) [span_89](start_span)It is volatile, meaning its contents are lost when the power is off.[span_89](end_span) [span_90](start_span)Types include Static RAM (SRAM) and Dynamic RAM (DRAM).[span_90](end_span)
    • [span_91](start_span)[span_92](start_span)
  • ROM (Read Only Memory): A non-volatile memory where information is stored permanently and can only be read, not written to.[span_91](end_span)[span_92](end_span) [span_93](start_span)It stores startup instructions.[span_93](end_span) [span_94](start_span)Variations include PROM (Programmable ROM) and EPROM (Erasable Programmable ROM).[span_94](end_span)
