  • Second Generation (1956-1963): Based on transistors and magnetic core storage[span_27](end_span).     • [span_31](start_span)
  • Third Generation (1964-1971): Used Integrated Circuits (ICs), or chips, which placed miniaturized transistors on silicon[span_31](end_span). [span_32](start_span)This drastically increased speed and efficiency[span_32](end_span). [span_33](start_span)The development of minicomputers took place during this era[span_33](end_span).
    • [span_34](start_span)[span_35](start_span)
  • Fourth Generation (1971-Present): Started with Large Scale Integration (LSI) and Very Large Scale Integration (VLSI) circuits, leading to the microprocessor[span_34](end_span)[span_35](end_span). [span_36](start_span)This allowed for the creation of the personal computer (PC)[span_36](end_span).
    • [span_37](start_span)[span_38](start_span)
  • Fifth Generation (Present and Beyond): Based on Artificial Intelligence (AI) and parallel processing[span_37](end_span)[span_38](end_span). [span_39](start_span)[span_40](start_span)These computers are still in development, with applications like voice recognition and neural networks[span_39](end_span)[span_40](end_span).

    [span_78](start_span)[span_79](start_span)

3.3 MEMORY HIERARCHY AND THEIR NEEDS

[span_96](start_span)

To balance speed, size, and cost, modern computers use a memory hierarchy[span_96](end_span). [span_97](start_span)This hierarchy includes fast but small CPU registers at the top, followed by cache memory, main memory, and finally large but slow secondary storage[span_97](end_span). [span_98](start_span)The overall goal is to achieve the highest possible access speed while minimizing the total cost of the memory system[span_98](end_span).
